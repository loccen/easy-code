项目描述

这是一个基于 React 18 + TypeScript + Ant Design 构建的现代化管理后台系统。 系统采用最新的前端技术栈，提供了完整的后台管理功能模块。

主要功能

  • 用户管理：用户列表、角色权限、部门管理
  • 系统管理：菜单管理、字典管理、参数配置
  • 数据统计：图表展示、数据分析、报表导出
  • 日志管理：操作日志、登录日志、系统监控

技术特色

  • 响应式设计，支持多种设备访问
  • 组件化开发，代码复用性高
  • TypeScript 类型安全
  • 完整的权限控制系统
  • Docker 容器化部署

技术栈

React 18 TypeScript Ant Design Redux Toolkit Vite Docker Nginx
